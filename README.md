# 🎯 Interview Trainer AI

An AI-powered Interview Preparation Platform built using **IBM watsonx.ai**, **FastAPI**, **React (Vite)**, **MongoDB**, and **ChromaDB** to help students and professionals prepare for technical and HR interviews through personalized question generation, AI evaluation, and Retrieval-Augmented Generation (RAG).

> Developed as part of the **IBM SkillsBuild x Edunet Foundation AI & Cloud Internship 2026**.

---

## 🚀 Live Demo

🌐 Frontend: https://interview-trainer-ai-three.vercel.app

📂 GitHub Repository: https://github.com/janhavi-2011/Interview-Trainer-AI

# ✨ Features

- 🔐 Secure User Authentication (JWT)
- 🤖 AI-powered Interview Question Generation
- 💬 HR, DSA, Behavioral & System Design Preparation
- 🧠 RAG (Retrieval-Augmented Generation) based Interview Assistant
- 📚 Knowledge Base powered by ChromaDB
- 📈 Personalized Learning Roadmap
- 📊 Interview History Tracking
- ⭐ AI-based Answer Evaluation
- 🏢 Company-specific Interview Preparation
- 📱 Responsive Modern UI

# ⚙️ System Workflow

1. User Login / Registration
2. Select Interview Type
3. Generate AI Questions
4. Retrieve Relevant Knowledge (RAG)
5. IBM Granite Model Generates Response
6. Evaluate Answers
7. Save Interview History
8. Display Personalized Roadmap
